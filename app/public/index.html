OFAC Sanctioned Entities Search

How to Use

This page allows you to search the Office of Foreign Assets Control (OFAC) sanctioned entities list. Type your query into the Query search box and then click the Search button to get a list of results. To search a specific field, type that field into the Field search box. If left blank, the search will be done across all fields. Note that some of the fields are nested, so you will need to use dot notation to access them. For example, to search a country, use addressList.address.country. There is also a Kibana instance for more detailed exploration of the Elasticsearch index created for this page. On the first visit to the Kibana page, you will have to set the sdnentries index as the default index. You will be prompted to set a default index when visiting the Discover tab.
